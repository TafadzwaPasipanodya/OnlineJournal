View Notes

Title: View Notes

Actor: User

Prerequisite: The user must be logged into a valid account before they can access this feature.

Action: CLick the view notes tab on the menu bar.

Effect: This will show the user all of the notes that they have made in order of when they were created. If the user hasn't created ay notes yet, they will just be notified that they don't have any notes in their queue.
