Register

Title: Register

Actor: User

Prerequisite: The user must not already have an account with this web application.

Action: Click on the 'not a member' link.

Effect: Redirects user to a page where they are asked to input a username and password and if the username isn't already taken by another user will then create a user with the given username and password.
