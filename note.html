Note

Title: Note Tab

Actor: User

Prerequisite: The user must be logged into a valid account before they can access this tab.

Action: Click the note tab in the navigation bar.

Effect: Redirects the user to the note taking page, where they are able to access old notes and create new ones. They are also able to delete notes they have created and move notes into specific course folders.

Next: None
