Search Courses

Title: Search Courses

Actor: User

Prerequisite: The user must be logged into a valid account before they can access this feature.

Action: The user will type in a course title into the search bar and click the search button.

Effect: The user will be informed of whether or not that course is already in the system, and if not will have the option to create that specific course.
